<h2>About</h2>  
A simple command-line Blackjack game written in Java based on the SYST17796 starter code.  
Players can place bets, choose Hit or Stand, and play against an automated dealer.

<h2>How to run</h2>  
1. Open the project in your IDE (NetBeans, IntelliJ, Eclipse).  
2. Make sure a JDK is selected (Java 17+).  
3. Run the file:  
   **ca.sheridancollege.project.PlayGame**  
4. The game will start in the console.

<h2>How to play</h2>  
1. Choose from the main menu:  
   - 1 - Play a round  
   - 2 - View basic winning strategy  
   - 3 - Exit  
2. When playing a round:  
   - Enter your bet  
   - Choose either to hit (take a risk) or stand (keep your card)
3. The dealer then plays automatically.  
4. The winner is shown and your balance updates.  
5. Play as many rounds as you like until you exit.
